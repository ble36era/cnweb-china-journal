# cnweb-china-journal

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容涉及中文网络环境下的各类信息记录与整理，不针对任何单一域名或具体网站。

## 项目简介

本仓库的主要目的是以 HTML 格式保存和展示一系列独立的网页内容。这些页面可能包含信息汇总、资料整理或静态展示，供有需要的研究者、开发者或普通用户参考使用。

所有页面均为独立文件，彼此之间无强依赖关系，可直接在浏览器中打开查看。

## 目录说明

```
.
├── README.md               # 本说明文件
├── index.html              # 仓库首页/导航页（可选）
├── pages/                  # 存放各独立 HTML 页面
│   ├── page-example-1.html
│   ├── page-example-2.html
│   └── ...
└── assets/                 # 公共资源（如 CSS、JS、图片等）
    ├── style.css
    └── ...
```

- `pages/` 目录下存放所有独立的 HTML 页面。
- `assets/` 目录存放页面共享的样式、脚本或图片资源。
- 各页面可根据需要自行引用 `assets/` 中的文件。

## 页面归档说明

- 每个页面均为独立完整的 HTML 文件，可直接在浏览器中运行。
- 页面内容可能包含文本、表格、链接、图片等元素，所有资源尽量内嵌或引用自 `assets/` 目录。
- 页面之间不互相链接（除非特别说明），不构成复杂网站结构。
- 归档内容可能不定期更新、补充或移除。
